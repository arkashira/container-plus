 # Partner Integration Roadmap for container-plus

This roadmap outlines the integration of key SaaS/APIs to enhance the functionality and value of our new product, container-plus. The following partners have been selected based on their relevance to the container ecosystem, potential for value-add, and the opportunity for affiliate/revenue-share.

## Partner Integrations (Priority: High to Low)

1. **Docker Hub** ([docker.com](http://docker.com))
   - Free-tier limits: Unlimited public repositories, 200 private repositories, 3,000 builds per month
   - Integration effort: Large
   - Value-add: Seamless container image management and distribution, improving the developer experience

2. **Google Cloud Build** ([cloud.google.com/build](http://cloud.google.com/build))
   - Free-tier limits: 120 minutes of build time per month, 1 concurrent build per project
   - Integration effort: Large
   - Value-add: Automated container build and deployment, reducing manual effort and potential errors

3. **JFrog Artifactory** ([jfrog.com/products/artifactory/](http://jfrog.com/products/artifactory/))
   - Free-tier limits: 2 concurrent users, 2GB storage
   - Integration effort: Medium
   - Value-add: Centralized management and distribution of container images, improving security and version control

4. **AWS Elastic Container Registry** ([aws.amazon.com/ecr/](http://aws.amazon.com/ecr/))
   - Free-tier limits: 100,000 pull requests, 100,000 push requests, 100,000 images per month
   - Integration effort: Large
   - Value-add: Secure and scalable container image management, integrating well with other AWS services

5. **GitLab Container Registry** ([about.gitlab.com/features/container_registry/](http://about.gitlab.com/features/container_registry/))
   - Free-tier limits: Unlimited private repositories, 2,000 builds per month, 10 concurrent builds
   - Integration effort: Medium
   - Value-add: Integrated container registry and CI/CD pipeline, improving developer productivity

6. **Red Hat OpenShift** ([redhat.com/en/technologies/cloud-native-application-platform/openshift](http://redhat.com/en/technologies/cloud-native-application-platform/openshift))
   - Free-tier limits: Limited to 3 developers, 1 project, 100GB storage
   - Integration effort: Large
   - Value-add: Enterprise-grade Kubernetes platform, offering advanced features for container orchestration and management

7. **Harbor** ([goharbor.io](http://goharbor.io))
   - Free-tier limits: 100 users, 100 repositories, 100GB storage
   - Integration effort: Medium
   - Value-add: Open-source container registry and security scanner, improving container security and compliance

These integrations will help container-plus to provide a comprehensive solution for container management, addressing the needs of developers and DevOps teams, while also offering opportunities for revenue-sharing through affiliate partnerships.
